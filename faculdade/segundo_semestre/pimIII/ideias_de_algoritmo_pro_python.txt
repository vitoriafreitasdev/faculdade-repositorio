Um paciente que responde de forma extremamente positiva ou negativa a um tratamento, fugindo do padrão observado nos demais participantes. 
Um sinal vital de um paciente (como frequência cardíaca ou pressão arterial) que se desvia consideravelmente dos parâmetros normais. Erros na coleta de dados, como um registro de dosagem de medicamento drasticamente diferente do esperado. )

== Ideia de como vamos construir o algoritmo de Python ==

rotinas de aprendizado de maquina para fazer as analises dos remédios e pacientes 

Vai utilizar:

arvore binaria AVL (arvore balanceada) => estruturar os dados que vão alimentar o Machine Learning(aprendizado de maquina), que vai ser feito com arvore de decisão.

se possível => lista de prioridades, remédio com prioridades maiores vão ser analisados primeiro 
ser for seguro de administrar, sem efeito colateral, eficaz. => 
passa para fase dois

se for seguro de administrar, com efeito colateral(leve), porem suportável, eficaz => passa para fase 2

seguro, efeito colateral (alto),  eficaz => precisa de reajustes.

não é seguro, efeito colateral alto insuportável, não eficaz => não passa. 

DFS => para pegar todos os remédios que foram para o teste
BFS => para achar apenas o remédios que passaram no teste
